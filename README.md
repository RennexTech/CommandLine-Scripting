## 📦 What's in the Box?

I teach what you need to know in batch scripting, enough to do `malware analysis/reverse engineering`.

`cmd` is fast, works in safe mode and replaces powershell when it gets blocked or when you forget syntax.  

## 🛠️ Requirements

- Windows (duh)
- A keyboard
- Occasional internet connection to Google why a command didn’t work

## The book ends with a warning to run the programs you write inside a sandbox, coz you can nuke your entire computer or registry.
